# AutoFlow AI 🚀

## Problem
Manual enterprise workflows are slow and error-prone.

## Solution
Multi-agent AI system that automates workflows end-to-end.

## Features
- Multi-agent orchestration
- AI-based decision making
- Self-healing workflows
- Audit trail

## Tech Stack
- FastAPI
- React
- SQLite / PostgreSQL

## Run Project

### Backend
cd backend
uvicorn app.main:app --reload

### Frontend
cd frontend
npm start
